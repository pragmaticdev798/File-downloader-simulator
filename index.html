Key Concepts I have used here

This Project looks simple but makes use of Javascript's Async concepts like,Promises,Fetch API etc.
first the URL is Fetched (url validation) if the response is received ok,using Async function which returns a Promise,
we then know the content-length from content length header,
then we get a reader from the response body stream with response.body.getReader()
we use a while loop to read through the chunks of data we received,
the read method returns an object {done,value} each chunk is stored in the chunks array,
download percentage is then calculated based of this information,
we then create Blob from returning data, so why do we create a blob?
because the returning data could be an image (raw data basically),
if it were a json response then we would have used .json instead of using a blob
